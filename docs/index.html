Driveboard Element

A simple yet powerful CNC controller for lasers and mills.

About Setup Buy View on GitHub

Driveboard Element is a fully integrated CNC controller with a beautiful user interface. It's a highly capable yet simple device for precise and robust control of 3-axis mills and laser cutters. The accompanying DriveboardApp is fully cross-platform (Linux, Windows, OSX, x86, ARM) and recognizes the hardware automatically. Unlike some other solutions, Driveboard Element is robust in noisy environments by virtue of its fully shielded case and opto-isolated wiring.

Specs

This project is in Beta phase. Development began in late 2015 and we started flight testing the system in 2016. We hope to conclude the beta phase by the end of 2016 (LOWI gets most mountain waves in fall). Our latest flight was up to FL150 and the system performed as designed. Yet for a very complete system we still have work to do:

  • 3-axis
  • 1-3A stepper motors
  • 30 kHz
